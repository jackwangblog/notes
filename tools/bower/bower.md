## Bower
[Bower](http://bower.io) 是 Twitter 出的一个包管理工具。让 WEB 开发者只需要使用几条简单的命令就可以管理想要使用的各种库或脚本，以及它们之间的依赖关系。

#### 安装
   	
``` 
	npm install -g bower
```	

#### 使用（以使用bootstrap为栗）
1. 首先可以查看下这个包的信息

```
	bower info bootstrap
```
